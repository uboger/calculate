# 简单的网页计算器实现过程
### 一、文件夹结构
- calculate（根目录）
- script（js文件存放目录）
- style（样式文件夹和ui图片夹）
- css（样式文件）
- image（ui图片文件）
- index.html 
#### 目录结构
    calculate/
              script/
                    calculator.js/
                    jquery-1.4.2.js/
              style/
                    css/
                        style.css/
                    image/
                        ui图片文件/
              index.html/
